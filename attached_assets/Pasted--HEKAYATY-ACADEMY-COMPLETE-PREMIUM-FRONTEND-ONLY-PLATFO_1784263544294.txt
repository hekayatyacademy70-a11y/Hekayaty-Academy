# HEKAYATY ACADEMY — COMPLETE PREMIUM FRONTEND-ONLY PLATFORM PROMPT

Create a world-class, production-ready, frontend-only web application called **Hekayaty Academy (أكاديمية حكاياتي)**.

This is not a simple course platform.

It is a complete Arabic educational ecosystem for discovering, training, developing, publishing, and monetizing writers, storytellers, illustrators, young talents, and creative professionals.

The platform must feel like a premium combination of MasterClass, Coursera, Udemy, Skillshare, Duolingo, Notion, and a modern publishing ecosystem, while maintaining a unique literary identity inspired by Hekayaty.

---

## GENERAL REQUIREMENTS

Build a frontend-only application.

Use:

* React
* TypeScript
* Tailwind CSS
* Vite
* React Router
* Framer Motion
* Shadcn UI
* Lucide Icons

The entire project must be structured as if it is ready for backend integration later.

All data should be mocked using realistic JSON objects, local state management, and service abstraction layers.

No backend implementation.

No API implementation.

No database implementation.

Frontend architecture must simulate real production behavior.
